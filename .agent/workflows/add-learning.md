---
description: How to add a new learning
---
When adding a new learning to this repository, you must classify it into a topic folder. 

1. Determine the topic of the learning (e.g., `python`, `git`, `javascript`).
2. Check if a folder named after the topic exists in the root of the repository.
3. If the folder does not exist, create it.
4. Add the learning file into that specific topic folder.
5. Do not add learning files directly to the root of the repository unless it doesn't fit any topic at all, which is extremely rare.
