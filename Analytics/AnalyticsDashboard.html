WINDOW FUNCTIONS 7-DAY ATTRIBUTION OPTIMIZED JOIN

Funnel Conversion: View to Cart

Tracks users who transitioned from viewing a product to adding it to their cart within 7 days, segmented by country, gender, and age cohorts.

WITH product_views AS ( SELECT distinct_id, timestamp AS view_time, JSON_EXTRACT_SCALAR(properties, '$.product_id') AS product_id FROM `project.dataset.events` WHERE event_name = 'Product Viewed' ), product_adds AS ( SELECT distinct_id, timestamp AS add_time, JSON_EXTRACT_SCALAR(properties, '$.product_id') AS product_id FROM `project.dataset.events` WHERE event_name = 'Product Added' ), joined_funnel AS ( SELECT v.distinct_id, v.product_id, v.view_time, MIN(a.add_time) AS first_add_after_view FROM product_views v LEFT JOIN product_adds a ON v.distinct_id = a.distinct_id AND v.product_id = a.product_id AND a.add_time BETWEEN v.view_time AND TIMESTAMP_ADD(v.view_time, INTERVAL 7 DAY) GROUP BY 1, 2, 3 ) SELECT u.country, u.gender, CASE WHEN u.age < 18 THEN 'Under 18' WHEN u.age BETWEEN 18 AND 35 THEN '18-35' ELSE 'Over 35' END AS age_group, COUNT(DISTINCT f.distinct_id) AS views, COUNT(DISTINCT CASE WHEN f.first_add_after_view IS NOT NULL THEN f.distinct_id END) AS additions FROM joined_funnel f JOIN `project.dataset.users` u ON f.distinct_id = u.distinct_id GROUP BY 1, 2, 3;

Industry Perspective

Critical for identifying friction in the user journey. By segmenting by age and country, brands can tailor UI experiences specifically for underperforming demographics.
COHORT ANALYSIS 30-DAY RETENTION

Repeat Purchase (Retention)

Identifies if a new customer returns for a second purchase within their first 30 days. Uses first-touch attribution logic.

WITH first_purchases AS ( SELECT distinct_id, MIN(timestamp) AS first_purchase_time FROM `project.dataset.events` WHERE event_name = 'Order Completed' GROUP BY 1 ), subsequent_actions AS ( SELECT e.distinct_id, e.timestamp AS return_time FROM `project.dataset.events` e JOIN first_purchases f ON e.distinct_id = f.distinct_id WHERE e.timestamp > f.first_purchase_time AND e.timestamp <= TIMESTAMP_ADD(f.first_purchase_time, INTERVAL 30 DAY) ) SELECT FORMAT_TIMESTAMP('%Y-%m', f.first_purchase_time) AS cohort, u.country, COUNT(DISTINCT f.distinct_id) AS base_users, COUNT(DISTINCT s.distinct_id) AS retained_users FROM first_purchases f LEFT JOIN subsequent_actions s ON f.distinct_id = s.distinct_id JOIN `project.dataset.users` u ON f.distinct_id = u.distinct_id GROUP BY 1, 2;

Industry Perspective

The "Life Blood" metric for DTC brands. High CAC (Customer Acquisition Cost) requires high retention for profitability. This query identifies "Golden Cohorts" for targeted marketing.
BASKET ANALYSIS PRODUCT AFFINITY

Product Affinity Analysis

Calculates the frequency of product pairings within the same order to power recommendation engines like 'Users also bought'.

WITH order_items AS ( SELECT JSON_EXTRACT_SCALAR(properties, '$.order_id') AS order_id, JSON_EXTRACT_SCALAR(properties, '$.product_name') AS p_name FROM `project.dataset.events` WHERE event_name = 'Order Completed' ) SELECT a.p_name AS prod_a, b.p_name AS prod_b, COUNT(*) AS pair_frequency FROM order_items a JOIN order_items b ON a.order_id = b.order_id AND a.p_name < b.p_name GROUP BY 1, 2 HAVING pair_frequency > 5 ORDER BY pair_frequency DESC;

Industry Perspective

Directly influences AOV (Average Order Value). By knowing what pairs well, merchandisers can bundle products or trigger dynamic cross-sell popups for higher revenue.
