# Learning Resources

This repository is a structured space for all learning materials, notes, and code snippets. It serves as a central hub for backtracking and revision.

## Repository Structure

- `/SQL`: Learning materials related to SQL.
- `/Python`: Learning materials related to Python.
- `/ETL`: Notes and examples on ETL processes.
- `/Analytics`: Insights and learning from analytics tools (e.g., GA4, Snowplow).
- `/Others`: Miscellaneous topics.

## How to Use

Add any new learning item into its respective category folder. If a new category is needed, create a new folder for it.
